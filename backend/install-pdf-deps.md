# Installation des dépendances PDF

Exécuter dans le dossier pointage-back :

```bash
npm install puppeteer handlebars
npm install --save-dev @types/handlebars
```

## Dépendances installées :
- **puppeteer**: Génération PDF à partir de HTML
- **handlebars**: Moteur de templates pour variables dynamiques
- **@types/handlebars**: Types TypeScript pour Handlebars
